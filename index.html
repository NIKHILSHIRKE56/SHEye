SHEye – AI-Powered Women’s Safety System

Real-time CCTV monitoring using AI to detect harassment, stalking, and accidents

👁 About SHEye

SHEye is an AI system designed to monitor CCTV footage in real-time and instantly detect signs of harassment, violence, stalking, or victim collapse. Built using YOLOv8, OpenCV, FastAPI, and Python, it proactively sends alerts via WhatsApp/SMS to concerned authorities — without human delay.

🧠 Tech Stack

  • 🔹 YOLOv8 for object and behavior detection
  • 🔹 OpenCV for video stream processing
  • 🔹 FastAPI for real-time backend API
  • 🔹 Python, Docker, and Linux for scalable deployment
